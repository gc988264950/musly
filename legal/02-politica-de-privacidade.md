## 2. Quais Dados Coletamos

### 2.1 Dados de Professores (Titulares de Conta)

| Categoria | Dados | Finalidade |
|---|---|---|
| Identificação | Nome completo, endereço de e-mail | Criação e autenticação de conta |
| Perfil profissional | Nome artístico/estúdio, instrumentos ensinados | Personalização do serviço |
| Dados de acesso | Registros de login, endereço IP, dispositivo, navegador | Segurança, prevenção de fraude, auditoria |
| Dados financeiros | Informações de transação, dados de assinatura | Cobrança e suporte a pagamentos |
| Conteúdo pedagógico | Planejamentos, anotações, arquivos enviados | Prestação do serviço |
| Dados de uso | Funcionalidades acessadas, tempo de uso, eventos na plataforma | Melhoria do produto, suporte técnico |

### 2.2 Dados de Alunos (Portal do Aluno)

| Categoria | Dados | Finalidade |
|---|---|---|
| Identificação | Nome completo, endereço de e-mail | Autenticação no portal |
| Dados acadêmicos | Histórico de aulas, tarefas, progresso, repertório | Prestação do serviço ao professor |
| Dados de acesso | Registros de login, endereço IP, dispositivo | Segurança |

**Atenção sobre menores:** Se um aluno tiver menos de 18 anos, o professor-responsável declara, ao criar o acesso, ter obtido o consentimento do responsável legal do menor. A Musly não coleta dados de menores diretamente e não verifica a idade dos usuários do Portal do Aluno — essa responsabilidade é do professor que cria o acesso.

### 2.3 Dados Coletados Automaticamente

- **Cookies e tecnologias similares:** conforme descrito em nossa [Política de Cookies]
- **Logs de servidor:** data, hora, endereço IP, tipo de requisição, código de resposta — retidos por até 90 dias
- **Dados de desempenho:** erros, tempos de carregamento — usados exclusivamente para estabilidade técnica

## 3. Bases Legais para o Tratamento

Nos termos do art. 7º da LGPD, o tratamento dos dados pessoais na Musly ocorre com fundamento nas seguintes bases legais:

| Base Legal | Dados Tratados |
|---|---|
| **Execução de contrato** (art. 7º, V) | Dados necessários para prestar o serviço contratado (conta, alunos, aulas, pagamentos) |
| **Legítimo interesse** (art. 7º, IX) | Dados de segurança, prevenção de fraude, logs técnicos, melhorias do produto |
| **Cumprimento de obrigação legal** (art. 7º, II) | Dados fiscais e financeiros exigidos por lei |
| **Consentimento** (art. 7º, I) | Cookies não essenciais, comunicações de marketing (quando aplicável) |

## 4. Como Usamos os Dados

### 4.1 Prestação do Serviço

- Autenticar e manter as contas de usuário
- Exibir dados do estúdio no painel do professor
- Disponibilizar o Portal do Aluno com seus dados pedagógicos
- Processar pagamentos e manter o histórico de assinaturas

### 4.2 Assistente Musly IA

Ao utilizar o Assistente Musly IA, os dados do seu estúdio (nomes de alunos, histórico de aulas, tópicos, tags de desempenho) são enviados a um modelo de linguagem de terceiro (atualmente **OpenAI**) para geração da resposta. Essa transmissão é feita de forma segura, exclusivamente para gerar a resposta solicitada, e está sujeita à [política de privacidade da OpenAI](https://openai.com/privacy).

**Não utilizamos os dados dos usuários para treinar modelos de IA** nem compartilhamos dados pedagógicos com terceiros além do estritamente necessário para o funcionamento do serviço.

### 4.3 Comunicação

Utilizamos o e-mail cadastrado para:
- Notificações operacionais essenciais (alterações de plano, avisos de segurança)
- Lembretes e alertas configurados pelo próprio usuário na plataforma
- Comunicações sobre atualizações relevantes dos Termos ou desta Política

Comunicações de marketing (novidades, promoções) são enviadas apenas com consentimento prévio e podem ser canceladas a qualquer momento.

### 4.4 Segurança e Conformidade

- Detectar e prevenir acessos não autorizados
- Investigar incidentes de segurança
- Cumprir obrigações legais e responder a ordens judiciais

## 5. Compartilhamento de Dados

A Musly não vende dados pessoais. Compartilhamos dados somente nas seguintes situações:

| Destinatário | Dados | Motivo |
|---|---|---|
| **Supabase** (infraestrutura) | Todos os dados armazenados | Banco de dados e autenticação |
| **OpenAI** (IA) | Dados pedagógicos do estúdio ao usar o Assistente IA | Geração de respostas da IA |
| **Cakto** (pagamentos) | E-mail, dados de transação | Processamento de pagamentos |
| **Vercel** (hospedagem) | Dados de tráfego | Hospedagem da aplicação |
| **Autoridades legais** | Conforme exigido | Cumprimento de ordens judiciais ou legais |

Todos os subprocessadores são contratados com cláusulas de proteção de dados adequadas.

## 6. Transferência Internacional de Dados

Alguns de nossos subprocessadores (Supabase, OpenAI, Vercel) estão localizados fora do Brasil, principalmente nos Estados Unidos. Essas transferências são realizadas com base em garantias contratuais adequadas e em conformidade com o art. 33 da LGPD.

## 7. Retenção de Dados

| Dados | Prazo de Retenção |
|---|---|
| Dados de conta ativa | Enquanto a conta estiver ativa |
| Dados após encerramento da conta | 30 dias disponíveis para exportação, depois excluídos |
| Dados financeiros e fiscais | 5 anos (obrigação legal) |
| Logs de acesso e segurança | 90 dias |
| Dados de menores | Excluídos conforme acima; professor deve conservar o consentimento |

Após os prazos acima, os dados são excluídos de forma segura ou anonimizados.

## 8. Segurança dos Dados

Adotamos medidas técnicas e organizacionais para proteger os dados pessoais, incluindo:

- Criptografia em trânsito (TLS/HTTPS) e em repouso
- Controles de acesso baseados em função (Role-Based Access Control)
- Autenticação segura via Supabase Auth
- Políticas de Row Level Security no banco de dados
- Monitoramento de acessos e logs de auditoria
- Resposta a incidentes conforme nossa [Política de Resposta a Incidentes]

Nenhum sistema é 100% seguro. Em caso de incidente que comprometa dados pessoais com risco relevante, notificaremos os titulares afetados e a Autoridade Nacional de Proteção de Dados (ANPD) nos prazos legais.

## 9. Direitos dos Titulares

Nos termos do art. 18 da LGPD, você tem os seguintes direitos em relação aos seus dados pessoais:

| Direito | Descrição |
|---|---|
| **Confirmação e acesso** | Saber se tratamos seus dados e acessá-los |
| **Correção** | Atualizar dados incompletos, inexatos ou desatualizados |
| **Anonimização, bloqueio ou eliminação** | Para dados desnecessários, excessivos ou tratados em desconformidade |
| **Portabilidade** | Receber seus dados em formato estruturado |
| **Eliminação** | Solicitar a exclusão de dados tratados com base em consentimento |
| **Revogação do consentimento** | Retirar o consentimento a qualquer momento |
| **Oposição** | Opor-se ao tratamento em caso de descumprimento da LGPD |
| **Informação sobre compartilhamento** | Saber com quem seus dados são compartilhados |

Para exercer esses direitos, acesse nosso [Canal LGPD] ou envie solicitação ao e-mail **privacidade@musly.com**. Responderemos em até 15 dias úteis.

A Musly pode solicitar verificação de identidade antes de atender a solicitações de titulares.

## 11. Dados de Menores
